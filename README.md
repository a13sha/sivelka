# Мастер по Климату

Статический одностраничный сайт для мастера по установке и обслуживанию кондиционеров в Москве. Собран на генераторе статических сайтов [Hugo](https://gohugo.io/) с использованием темы [hugo-scroll](https://github.com/sivaramsk/hugo-scroll).

## Стек технологий

| Компонент | Технология |
|-----------|------------|
| **Генератор** | [Hugo](https://gohugo.io/) (v0.115.2, расширенная версия) |
| **Тема** | [hugo-scroll](https://github.com/sivaramsk/hugo-scroll) — одностраничная тема с навигацией по секциям |
| **Шрифты** | Google Fonts: [Inter](https://fonts.google.com/specimen/Inter) (основной текст), [Manrope](https://fonts.google.com/specimen/Manrope) (заголовки) |
| **Иконки** | [Fork Awesome](https://forkaweso.me/) (форк Font Awesome 4) |
| **CSS** | Кастомные CSS-переменные в `variables.scss`, внешний `custom.css` (кэшируемый), медиазапросы, анимации (ключевые кадры) |
| **JavaScript** | jQuery 3.6.3, кастомные скрипты для переключения темы, плавного скролла, замены иконок |
| **Деплой** | [Netlify](https://www.netlify.com/) — автоматическая сборка из репозитория |

### Ключевые файлы

- **`hugo.toml`** — конфигурация проекта: язык (русский), параллакс-секции (`invertSectionColors = true`), отключение таксономий и RSS, SEO-мета-теги.
- **`netlify.toml`** — команда сборки `hugo --gc --minify -b $URL`, папка публикации `public`, версия Hugo 0.115.2.
- **`assets/css/variables.scss`** — CSS-кастомные свойства для двух тем: светлой (`data-theme="light"`) и тёмной (`data-theme="dark"`). Цвета подсветки, секций, футера, акцентных элементов.
- **`assets/css/variables.scss`** — цветовая палитра «Лёд и Медь» (Ice & Copper): light/dark темы.
- **`static/css/custom.css`** — кастомные стили сайта: компоненты (Service Cards, Advantage Grid, Timeline, Profile Card), анимации, wavy dividers, адаптивность. Вынесен из inline-стилей для кэширования.
- **`static/js/index.js`** — скрипт: плавный скролл к секциям, отслеживание активного пункта навигации, подмена плейсхолдеров иконок (`@fa-...@` → `<i class="fa ...">`). Scroll-обработчик с throttling через `requestAnimationFrame`.

## Удалённые зависимости

- `fonts.css` и 70 локальных файлов шрифтов (Open Sans, Oswald, Roboto Slab) — заменены на Google Fonts (Inter + Manrope).
- `css-vars-ponyfill.min.js` — IE11 polyfill, неактуален в 2026.

## Функциональность

### Секции сайта

Страница состоит из hero-обложки и следующих секций, каждая — отдельный Markdown-файл в `content/homepage/`:

1. **Добро пожаловать** — вступительный текст с приветствием
2. **Мои услуги** — две карточки: "Обслуживание кондиционеров" и "Установка кондиционеров" (SVG-иконки, анимация при наведении)
3. **Почему выбирают нас** — сетка 2×2 из пунктов (опыт, выезд, полный цикл, гарантия)
4. **Обо мне** — фото мастера (круглый портрет с обводкой) и текст об опыте
5. **Как мы работаем** — вертикальный таймлайн из 4 шагов (заявка → выезд → работы → сдача)
6. **Контакты** — email, телефон, регион (Москва и МО)

### Переключение тёмной темы

- Кнопка фиксирована в правом верхнем углу (значок солнца/луны).
- Выбор сохраняется в `localStorage`.
- При первом визите применяется системная цветовая схема (`prefers-color-scheme`).
- CSS-переменные переключаются через атрибут `data-theme` на `<html>`.

### Анимации и визуальные эффекты

- Фон шапки — анимированный градиент (`heroGradient`, 10 секунд, бесконечно).
- Плавающие иконки ❄ и 🔧 в hero-области (`floatIcon`).
- Секции появляются с анимацией `fadeInUp`.
- Кнопки с закруглением 50px, тенью и подъёмом при наведении.
- Карточки услуг с градиентной полоской сверху и подъёмом при hover.
- Боковая навигация (sticky) с размытием (`backdrop-filter: blur`).
- Волнообразный разделитель между секциями (SVG-mask).
- Круглый портрет с двойной обводкой и тенью.

### Доступность (Accessibility)

- Навигация с `role="navigation"` и `aria-label`.
- Активный пункт меню отмечен `aria-current="true"`.
- Иконка переключателя темы имеет `aria-hidden="true"`.
- Кнопки и ссылки имеют видимый `:focus-visible` outline.
- Контраст текста соответствует WCAG AA (4.5:1+).

### Производительность и SEO

- Ленивая загрузка изображений (`loading="lazy"`).
- CSS-вариабельные шрифты через Google Fonts (Inter, Manrope).
- Разметка JSON-LD (`LocalBusiness`) для поисковых систем.
- Open Graph / Twitter Cards / Schema.org микроразметка.
- Preconnect к Google Fonts.
- Атрибут `title_guard = false` в конфиге (прозрачный фон для заголовка).

## Установка и запуск

### Требования

- [Hugo Extended](https://gohugo.io/installation/) версии 0.115.2 или новее (требуется SCSS-поддержка).
- Git (для клонирования и субмодулей).

### Локальный запуск

```bash
# Клонировать репозиторий вместе с субмодулями
git clone --recurse-submodules https://github.com/ваш-username/sivelka.git
cd sivelka

# Запустить сервер разработки
hugo server -D
```

Сайт будет доступен по адресу `http://localhost:1313/`. Флаг `-D` включает черновики (draft-страницы).

### Сборка production-версии

```bash
hugo --gc --minify
```

Готовый сайт появится в директории `public/`.

## Деплой

Проект настроен на деплой через **Netlify**. Конфигурация в `netlify.toml`:

```toml
[build]
  command = "hugo --gc --minify -b $URL"
  publish = "public"

[build.environment]
  HUGO_VERSION = "0.115.2"
```

1. Подключите репозиторий к Netlify.
2. Netlify автоматически определит `netlify.toml` и выполнит сборку.
3. Переменная `$URL` подставляется Netlify автоматически (URL продакшена или preview).

## Кастомизация

### Цветовая схема: «Лёд и Медь» (Ice & Copper)

Палитра построена на контрасте холодного и тёплого:

- **Лёд** (`--highlight: #0891b2` / `#22d3ee`) — ассоциация с чистым кондиционированным воздухом, современными технологиями. Используется для брендовых элементов, иконок, рамок.
- **Медь** (`--warm-accent: #ea580c` / `#f97316`) — ассоциация с медными трубками кондиционеров, символом качественного монтажа. Используется для CTA-кнопок и акцентов.

Файл `assets/css/variables.scss` содержит CSS-переменные для обеих тем:

| Переменная | Light | Dark | Назначение |
|-----------|-------|------|------------|
| `--highlight` | `#0891b2` | `#22d3ee` | Основной акцент (кнопки, ссылки, активные элементы) |
| `--warm-accent` | `#ea580c` | `#f97316` | Тёплый акцент (CTA, медный оттенок) |
| `--section-light-bg-color` | `#f8fafc` | `#0b0f19` | Фон светлых секций |
| `--section-dark-bg-color` | `#0f172a` | `#1e293b` | Фон тёмных секций |
| `--section-uni-quote` | `#475569` | `#94a3b8` | Второстепенный текст (WCAG AA) |
| `--cover-text-color` | `#0f766e` | `#67e8f9` | Цвет текста на hero-обложке |

Чтобы изменить цветовую схему, отредактируйте значения в `:root` (светлая) и `:root[data-theme="dark"]` (тёмная) в `variables.scss`.

### Добавление новой секции

1. Создайте новый Markdown-файл в `content/homepage/` (например, `reviews.md`):

```yaml
---
title: "Отзывы"
weight: 5
navigation_menu_title: "Отзывы"
header_menu: true
---

Текст секции.
```

2. Если нужна кнопка в hero-области, добавьте `header_menu: true`.
3. Порядок секций определяется полем `weight`.
4. Секции с чётным/нечётным индексом автоматически получают тёмный или светлый фон (настраивается через `invertSectionColors` в `hugo.toml`).

### Изменение логотипа и фона

Логотип и заголовок hero-области задаются в `content/_index.md`:

```yaml
header_logo: "images/ac-icon.svg"
header_headline: "Мастер по Климату"
header_subheadline: "<b>Установка</b> & <b>Обслуживание</b> Кондиционеров"
```

Фоновое изображение можно добавить через `header_image` (сейчас пусто — используется CSS-градиент).

### Шрифты

Основные шрифты подключаются через Google Fonts в `layouts/partials/custom_head.html`. Для замены достаточно изменить ссылку на Google Fonts и соответствующие `font-family` в `static/css/custom.css`.
